Native desktop runtime generated by `scripts/prepare-desktop-runtime.py`.

Release builds populate this directory with:

- `pdfium/`: the platform PDFium shared library used to render scanned PDF pages.
- `ocr/bin/`: the Tesseract executable.
- `ocr/lib/`: private shared libraries required by Tesseract where applicable.
- `ocr/tessdata/`: pinned `vie` and `eng` language models.

Generated binaries and models are intentionally gitignored.

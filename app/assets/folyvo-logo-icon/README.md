# Folyvo — bộ nhận diện (bản chốt 5d: than + cam)

## Màu
- Cam chính: #FF7A1A (nhạt: #FFBA88, đậm: #F4560A)
- Than: #171B26 (gradient icon: #3A4150 → #171B26)
- Chữ trên nền sáng: Fol #171B26 · yvo #FF7A1A
- Chữ trên nền tối/cam: Fol #FFFFFF · yvo #FF7A1A
- Font wordmark: Inter Bold (700), letter-spacing ≈ -0.035em

## File
- app-icon.svg / favicon.svg — icon app vector (nền than, F cam)
- app-icon-{16…1024}.png — bộ icon raster (app desktop, web, PWA, favicon.ico nguồn)
- app-icon-light-{256,512}.png — biến thể nền trắng
- logo-mark-orange.svg/.png — mark F cam, nền trong suốt (dùng trên nền sáng/tối)
- logo-mark-white.svg/.png — mark F trắng, nền trong suốt (dùng trên nền cam/ảnh tối)
- logo-horizontal-on-dark.png / logo-horizontal-on-light.png — lockup ngang
- logo-stacked-on-dark.png / logo-stacked-on-light.png — lockup dọc

## Gợi ý dùng

- Favicon: favicon.svg + app-icon-32/16
